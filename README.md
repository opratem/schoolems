## 📋 Project Overview

**SchoolEMS** is a full-stack web application developed to streamline the management of school employee records and leave requests. It provides a centralized platform for administrators, HR personnel, and staff to efficiently manage and track employee data. This project was developed during my internship at **FlexiSAF** as a practical demonstration of full-stack development skills.

---

## 🎯 Objectives

- Provide an intuitive and responsive interface for managing employee information.
- Enable secure and role-based access to functionalities.
- Automate the leave request and approval process.
- Centralize employee and leave data for improved administration.

---

## 🛠️ Technologies Used

- **Frontend**: ReactJS, JavaScript, HTML, CSS
- **Backend**: Spring Boot (Java)
- **Database**: PostgreSQL
- **API Communication**: Axios
- **Authentication**: JSON Web Tokens (JWT)
- **State Management**: React Hooks (`useState`, `useContext`)
- **Testing**: Jest, React Testing Library

---

## 🚀 Key Features

### 👥 Employee Management
- Create: Add new employee records (name, employee ID, department, position, etc.)
- Read: View employee details in a list or detailed view
- Update: Edit employee information
- Delete: Remove employee records

### 📝 Leave Management
- Create: Employees submit leave requests (type, date range, reason)
- Read: Managers/Admins view and track leave requests
- Update: Approve or reject leave requests
- Delete: Employees can delete pending leave requests

### 🔒 Authentication & Authorization
- Secure login system using JWT
- Role-based access control (Admin, Manager, Employee)

### 💡 UX & Functionality
- Responsive and intuitive UI
- Dashboard for administrators
- Client-side validation for data integrity
- Real-time feedback and error handling
